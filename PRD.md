# nuevo — Product Requirements Document

**버전**: 0.4  
**최종 수정**: 2026-05-13  
**스코프**: MVP (창작자 온보딩 + 리믹스 포함)

---

## 1. 제품 개요

### 한 줄 정의
AI로 만든 인터랙티브 앱을 올리고, 유저가 피드에서 직접 체험하며, 창작자는 체험 기반 수익을 얻는 플랫폼.

### MVP 슬로건
> **"올리고 → 바로 놀아보는"**

창작자는 이미 AI로 앱을 만든다. nuevo는 그 앱을 **올리는 곳**이고, 유저는 **직접 실행하는 곳**이다.

### MVP 핵심 루프

```
[창작자] 가입 → AI 앱 URL 올리기 → 피드에 노출
[방문자] 피드 → 작품 탭 → 앱 즉시 체험 → 리믹스 → 내 버전 올리기
[원본 창작자] 리믹스 알림 → 파생 수익 확인
```

### 포지셔닝

| 플랫폼 | nuevo가 벤치마킹한 것 | nuevo가 다른 것 |
|--------|----------------------|----------------|
| Instagram | 피드 UX, 팔로우/피드/탐색/알림 구조 | 영상 → 직접 실행 가능한 앱 |
| YouTube | 창작자 수익 배분 (광고 + 구독) | 조회수 → 체험 시간 기반 정산 |

### 핵심 가설
> Instagram Reels에서 AI 앱을 "영상으로 찍어" 공유하는 컨텐츠가 폭발적으로 증가 중.  
> 창작자들이 앱 자체를 올릴 수 있다면, 유저는 영상을 보는 대신 직접 체험한다.  
> 직접 체험 = 더 높은 참여 = 더 많은 광고 가치 = 창작자에게 더 많은 수익.

## 2. 타겟 사용자

### 창작자 (Creators)
- **주요 타겟**: Claude Code, Cursor, Replit로 앱을 만드는 코딩 비전공자 / 바이브코더
- **행동 패턴**: 만든 앱을 영상으로 찍어 Instagram Reels에 올리는 중
- **니즈**: "영상 말고 앱 자체를 공유하고 싶다, 그리고 수익도 원한다"
- **예시 창작자**: @chavida_ai (손 제스처 꽃 앱, 좋아요 10.9k)

### 체험자 (Users)
- **주요 타겟**: AI 창작물에 관심 있는 20-35세, 새로운 인터랙티브 경험을 즐기는 사람
- **행동 패턴**: Instagram Reels에서 AI 앱 영상을 보고 "나도 해보고 싶다"고 생각
- **니즈**: "영상 말고 직접 해보고 싶다"

### 브랜드 (Advertisers / Sponsors)
- AI 네이티브 컨텐츠에 관심 있는 브랜드
- 유저가 직접 조작하는 인터랙티브 광고 = 높은 브랜드 recall

## 4. 앱 임베딩 스펙 (창작자 가이드)

창작자가 제출하는 앱은 아래 스펙을 충족해야 한다:

```
뷰포트: 390px (iPhone 기준), 높이는 자유
공개 접근: 인증 없이 접근 가능한 URL
iframe 허용: X-Frame-Options: ALLOWALL 또는 미설정
권장 플랫폼: Vercel, GitHub Pages, Glitch, CodePen, Replit
카메라/마이크: 사용 시 contentType 필드에 선언
```

nuevo는 창작자 앱을 아래 sandbox로 임베딩:
```
sandbox="allow-scripts allow-same-origin allow-forms allow-popups"
allow="camera; microphone"
```

## 5. 수익 모델

### 5.1 수익 구조

```
플랫폼 총 수익
├── 광고 수익 (Ads)           ~60%
│   ├── 체험 전 프리롤 광고 (6초 스킵 가능)
│   └── 체험 중 배너 (비침투적)
├── nuevo+ 구독 수익          ~30%
│   ├── 월 4,900원 (광고 제거 + 창작자 분석 도구)
│   └── 연 39,000원
└── 브랜드 스폰서십            ~10%
    └── 브랜드가 창작자에게 의뢰하는 인터랙티브 광고 앱
```

### 5.2 창작자 수익 배분

**YouTube와의 차이점:**
- YouTube: 조회수(View) 기반 — 영상을 1초만 봐도 카운트
- nuevo: **체험 세션(Experience Session)** 기반 — 앱을 실제로 실행하고 인터랙션한 시간 기반

**수익 분배 비율:**

| 티어 | 조건 | 창작자 : 플랫폼 |
|------|------|----------------|
| 일반 (미가입) | — | 수익 없음 |
| 새싹 파트너 | 월 500+ 체험 세션 | 60 : 40 |
| 성장 파트너 | 월 5,000+ 체험 세션 | 70 : 30 |
| 프로 파트너 | 월 50,000+ 체험 세션 | 75 : 25 |
| 챔피언 파트너 | 월 500,000+ 체험 세션 | 80 : 20 |

**가중 체험 점수 (WES) 계산:**

```
WES = (체험 세션 수 × 1.0)
    + (총 체험 시간(분) × 0.8)
    + (반응 수 × 1.5)
    + (댓글 수 × 2.0)
    + (리믹스 수 × 5.0)
```

### 5.3 리믹스 수익 공유

- 1단계 리믹스: 원본 창작자에게 리믹스 수익의 10% 추가 귀속
- 2단계 이상 리믹스: 원본 창작자 5%, 직전 창작자 5%

→ 창작 생태계 형성 인센티브. 리믹스 허용이 창작자에게도 이득.

### 5.4 브랜드 스폰서십

브랜드가 특정 창작자에게 인터랙티브 앱 제작을 의뢰:
- nuevo가 매칭 중개 (수수료 20%)
- 창작자는 앱을 만들고 nuevo에 올림
- 유저는 "스폰서 체험"으로 표시된 앱을 무료로 체험
- 브랜드는 체험 수 / 체험 시간 리포트를 받음

## 6. 크리에이터 파트너십 프로그램

### 가입 조건
1. 계정 생성 30일 이상
2. 작품 최소 3개 이상 게시
3. 최근 30일 체험 세션 500회 이상
4. 커뮤니티 가이드라인 위반 없음

### 혜택
- 수익 배분 참여
- 크리에이터 스튜디오 (상세 분석)
- 프로필 파트너 배지 ✦
- 브랜드 스폰서십 매칭 우선권
- 신기능 얼리 액세스

## 9. MVP 이후 확장 로드맵

### Phase 2 — "만들기" 기능
플랫폼 내 AI 생성 기능. 창작자가 새 앱을 만들 때 nuevo를 시작점으로 삼도록 한다.

| 기능 | 설명 |
|------|------|
| 인터랙티브 생성 | 프롬프트 → Claude/GPT가 React 앱 코드 생성 → 자동 배포 |
| 오디오 생성 | 프롬프트 → Udio/Suno API 연동 |
| 이미지 생성 | 프롬프트 → Midjourney/DALL-E 연동 |
| 리믹스 + AI | 기존 앱 코드 기반으로 AI가 변형 생성 |

### Phase 3 — 실 정산
- 체험 세션 실 트래킹 (iframe onLoad/unload)
- WES 서버사이드 계산 + 월별 정산
- 출금 API (토스페이먼츠 / 스트라이프)
